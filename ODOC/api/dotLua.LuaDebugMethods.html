The methods of the LuaDebug class are listed below. For a complete list of LuaDebug class members, see the LuaDebug Members topic.

Public Static Methods

FromFunction Construcs a new LuaDebug object containing information about a non-active function from the given table. A debug object from a non-active function does neither have a caller nor a level.
FromLevel Constructs a new LuaDebug object containing information about the given level. Level 0 is the current called function, thereas "level+1" is the level which called "level".
