LuaStack Overloaded. Initializes a new instance of the LuaStack class.

Public Instance Properties

Count Retrieves the amount of items on the stack. Since LUA starts counting at 1, the index of the top item and the amount of items on the stack is equal. Thus Count only returns the value of TopIndex.
Item Retrieves a given item from the stack.
TopIndex Sets or retrieves the index of the top level item. If set to 0 the entire stack is being deleted.
TopValue Retrieves the value of the top level item without removing it from the stack.
