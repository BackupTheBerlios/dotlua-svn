dotLua Namespace

Namespace hierarchy

Classes

Class Description
Lua The core class of the library. It provides public methods to manipulate a given LUA state object.
LuaDebug Interface for providing debug information.
LuaException Thrown if an error occures in the LUA subsystem. Any other exception thrown by dotLUA derives from this class.
LuaFunctionArgs A class which sums up high level functionality for callback routines.
LuaGC Contains all necessary operations to manipulate a states garbage collector.
LuaLibrary This attribute marks a class to be loaded as a library for LUA. A LUA library is a class which contains static methods. Any static method which fits the LuaCallbackFunction delegate is being associated to a public table named after the class. The version parameter helps a script to differ between versions of one and the same library.
LuaStack Allows the manipulation of a given stack frame.
LuaState Represents a Lua namespace state.
LuaSyntaxException Thrown if the LUA script has syntax errors.
LuaTable Represents a LUA table and provides operations to modify those.
LuaTables Contains a container for all tables in a given state.
LuaThread This class describes a couroutine as used by the LUA library.
System dotLUA standard library for LUA.

Delegates

Delegate Description
