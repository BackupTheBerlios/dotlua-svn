Lua.Register Method 

Registers a callback function. Every callback function is being registered in the global namespace.

public void Register(
   String name,
   LuaCallbackFunction function
);

Parameters

String
Name of the function, as it should be available inside the Script
LuaCallbackFunction
Application defined callback function

Remarks

To register a callback function in the given namespace, create a new variable holding a table. Then insert the callback function into this table to retrieve a new namespace.
