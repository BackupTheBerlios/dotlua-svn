The dotLua project:

dotLua is a C# class library which allows C# 2.0 to interact with the LUA scripting engine. It bases on the original, but slightly modified, LUA C API. LUA is a leightweight, fast, structual scripting language.

Core features:

  • Easy to use interface. But it retains the power of the native C LUA API.
  • C# callbacks for LUA: LUA calls C# and reverse.
  • You can also write LUA libraries in form of C# static classes.
  • Benefit from a small and lightweight .NET class library.
